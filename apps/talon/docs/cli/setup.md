# `talon setup`

Initialize `~/.talon/talon.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
talon setup
talon setup --workspace ~/.talon/workspace
```

To run the wizard via setup:

```bash
talon setup --wizard
```
