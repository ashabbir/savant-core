# Feature Guide

## Board
- Kanban columns from project settings
- Drag & drop tasks
- Grouping (priority/type/epic/assignee)

## Routes
- `/storyboard` story board view
- `/epicboard` epic board view
- `/profile` profile/settings
- `/users` admin users
- `/agents` agent management
- `/logout` clear session and return to storyboard

## Task Details
- Edit title/assignee/priority/type/tags
- Comments & activity history
- Assignee/Created By pickers show users only (agents excluded)

## Agents
- Agent registry (admin)
- Assignment triggers optional auto‑processing
- Routing rules (project‑scoped) for agent selection
- Agent chat author list is limited to users with a preferred agent attached

## Quotas & Usage
- Monthly token/cost limits at project or user level
- Talon runs record token/cost usage entries

## Notifications
- User‑level Slack webhook or email subscriptions
- Mentions trigger notifications on comments
- Task moves notify subscribers

## Agent Chat
- Threaded messages per agent

## Activity Feed
- Global activity log

## Documents
- List + preview
